# Databricks
Project 1:-

Key steps in the project:
Data Ingestion: The project starts by importing raw emission data from a GitHub repository into Databricks, renaming it to emissions data for clarity.

Data Cleaning and Transformation: Real-world data comes with issues. The instructor demonstrates using SQL queries to clean a string column containing characters into a numerical format for analysis.

Data Analysis: Several queries are written to extract insights, such as emissions per person by county, total emissions by state, and identifying the top 10 highest-emitting counties.

Visualization: The data is visualized using various tools, including a point map for geographical location data, a scatter plot to compare emissions versus population (18:01), and a pie chart to show emissions percentages).

Dashboard Creation: Finally, these visualizations are compiled into a cohesive United States emissions breakdown dashboard).
